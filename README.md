# Professional Web Developer Portfolio

A modern, responsive portfolio website showcasing web development skills and projects with smooth animations and excellent performance.

## Features

- **Modern Design**: Clean, professional layout with smooth animations
- **Fully Responsive**: Mobile-first design that works on all devices
- **Performance Optimized**: Fast loading with optimized images and code
- **Interactive Elements**: Hover effects, smooth scrolling, and micro-animations
- **Skills Showcase**: Comprehensive display of programming languages and frameworks
- **Project Gallery**: Featured projects with overlay effects and technology tags
- **Contact Form**: Functional contact form with validation
- **Social Links**: Easy access to professional social media profiles

## Setup Instructions

1. **Clone or download** the portfolio files
2. **Replace placeholder content** with your information:
   - Update name, contact details, and bio in `index.html`
   - Add your profile image as `profile-image.jpg`
   - Add project images (`project1.jpg`, `project2.jpg`, `project3.jpg`)
   - Update project descriptions and links
   - Modify social media links in the contact section

3. **Customize styling** (optional):
   - Adjust colors in `styles.css`
   - Modify animations and transitions
   - Update typography and spacing

4. **Deploy** to your preferred hosting platform:
   - GitHub Pages
   - Netlify
   - Vercel
   - Traditional web hosting
